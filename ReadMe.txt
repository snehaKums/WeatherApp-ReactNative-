About App:

User can search any place in the searchbar. On selecting the place the location will be shown in the map. On click of marker user can check the Weather details of the place like Temperature,wind speed, humidity and weather condition


How to run the app:

1.Clone/Download the app.
2.Run the command 'npm install' in command line.
3.Open emulator or connect device using usb.
4.Run command 'react-native run-android' to run the app on phone/device
5.Incase using IOS phone, run command 'pod install' inside ios folder after 'npm install' and run command 'react-native run-ios' to run on device.
